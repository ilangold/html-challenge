# html-challenge

### Summary
This repository contains 2 jupyter notebooks used to scrape data from [Mars News.com](https://static.bc-edx.com/data/web/mars_news/index.html) and [Mars Weather](https://static.bc-edx.com/data/web/mars_facts/temperature.html) and summarizes the leading news stories on Mars and annual Mars weather.

### Use
To scrape and summarize Mars news, open and run [part_1_mars_news.ipynb](https://github.com/ilangold/html-challenge/blob/main/part_1_mars_news.ipynb). To scrape and summarize Mars weather, open and run [part_2_mars_weather.ipynb](https://github.com/ilangold/html-challenge/blob/main/part_2_mars_weather.ipynb).
